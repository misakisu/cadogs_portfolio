# Animal Stay Cadogs

## サイト概要
大切な家族の一員であるペット専用のホテルサイトがあったらなと思ったことはありませんか？
Cadogsでは、利用者のコメントも確認でます。

### サイトテーマ
猫や犬専用のホテル・預かり者情報サイト

### テーマを選んだ理由
犬・猫を安心して預けることのできる場所をまとめて探すサイトがなく、作成してみようとおもいました。

### ターゲットユーザ
犬や猫を飼っているオーナー様

### 主な利用シーン
旅行時、不在時
