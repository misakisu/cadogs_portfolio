# Animal Stay Cadogs

## サイト概要
大切な家族の一員であるペット専用のホテルサイトがあったら良いなと思ったことをきっかけにこのサイトを作成いたしました。
Cadogsでは、以下の機能を活用してお客様、ホテルオーナー様、管理者様が安心してサイトを利用できます。
ホテルの掲載に関しては、ホテルオーナー様がホテルを申請した後、管理者による承認が行われ、申請が許可されたホテルのみ掲載されるため、お客様は安心してホテル検討をすることができます。

### 機能一覧
顧客/オーナー/管理者画面、ペットジャンル・ペット追加、ホテルいいね・コメント・お気に入り一覧、検索・お問い合わせ、新規登録・一覧・カテゴリ、ホテル新規登録・一覧・カテゴリ、予約機能、ジャンル検索、非同期通信、画面スライダー、ホテル検索、ホテル承認完了メール(ホテルオーナー様向け)

### サイトテーマ
猫や犬専用のホテル予約プラットフォーム

### テーマを選んだ理由
犬・猫を安心して預けることのできるホテルをまとめて探すことができるサイトがなく、作成してみたいと思いました。

### ターゲットユーザ
犬や猫を飼っているオーナー様、犬や猫向けのホテルを運用されているホテルのオーナー様

### 主な利用シーン
犬や猫を飼っているオーナー様の旅行時
