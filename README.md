# Animal Stay Cadogs

## サイト概要
大切な家族の一員であるペット専用のホテルサイトがあったら良いなと思いこのサイトを作成いたしました。
Cadogsでは、利用者のコメントを参考にしたり、お気に入りを登録を活用して、ペット専門のホテルを一つのプラットフォームで探すことができます。
ホテルオーナーがホテルを申請した後、サイト管理者による申請承認が行われ、申請が許可されたホテルのみ掲載されるため、お客様は安心してホテル検討をすることができます。

### サイトテーマ
猫や犬専用のホテル予約プラットフォーム

### テーマを選んだ理由
犬・猫を安心して預けることのできるホテルをまとめて探すことができるサイトがなく、作成してみたいと思いました。

### ターゲットユーザ
犬や猫を飼っているオーナー様、犬や猫向けのホテルを運用されているホテルのオーナー様

### 主な利用シーン
犬や猫を飼っているオーナー様の旅行時
